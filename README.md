
  # Autism Support Hub Website

  This is a code bundle for Autism Support Hub Website. The original project is available at https://www.figma.com/design/dKMNhHjtRjCQk9H9yLhk0y/Autism-Support-Hub-Website.

  ## Running the code

  Must install Node.js prior to running commands. (nodejs.org, LTS version)
  
  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
